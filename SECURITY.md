# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a vulnerability

Please open a private security advisory on GitHub or email the maintainer. Do not disclose kernel-driver or privilege-escalation issues publicly before a fix is available.

## Scope

- `discdpi` runs with administrator rights and loads WinDivert
- Only install releases from this repository with verified checksums
